# Prompt Intelligence Platform

A local AI-powered prompt optimizer built using:

- Python
- Streamlit
- Ollama
- Gemma 3

## Features

- Prompt Analysis
- Intent Detection
- Context Gap Detection
- Prompt Optimization
- Multiple Optimization Modes
- Local AI (No API Costs)

## Run

pip install -r requirements.txt

streamlit run app.py
